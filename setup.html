Setting up Nofri

Two parts: put your start page online, then install the focus extension.

Nofri has two pieces that work together:

  • Nofri Start — a distraction-free start page (Coptic date, saint of the day, a verse and prayer, your teams, a few trusted headlines).
  • Nofri Focus — a browser extension that gently blocks distracting sites and opens Nofri Start in every new tab.

Part A · Put Nofri Start online

This gives you one link that works on your computer and your phone.

  1. In your GitHub repository, open Settings → Pages.
  2. Under Source, choose “Deploy from a branch”, pick branch main and folder / (root), then Save.
  3. Wait about a minute. Your start page will be live at:
    …/start.html (on the published site this is your github.io address).
  4. Make it your home / startup page:
    • Chrome (desktop): Settings → On startup → “Open a specific page” → add the URL.
    • Phone: set it as your browser’s homepage, or add it to your home screen.

Once the Focus extension is installed, every new tab opens this page automatically too.

Part B · Install Nofri Focus

The extension runs from files on your own computer.

  1. On GitHub, click Code → Download ZIP and unzip it (or git clone the repo).
  2. Open chrome://extensions (works in Chrome, Edge, and Brave).
  3. Turn on Developer mode (top-right).
  4. Click Load unpacked and select the extension folder.
  5. Click the ☦ Nofri Focus toolbar icon → Settings: set your Nofri Start URL, then adjust the blocked sites and (optionally) a schedule.
  6. Try visiting twitter.com — you’ll get the calm Pause page. 🎉

Prefer a normal install on every device? Upload the provided nofri-focus.zip to the Chrome Web Store Developer Console (a one-time $5 fee). Listing text and screenshots are included in the extension/ folder.

Tips

  • Teams: in Nofri Start → Customize, just type a team name (e.g. “Arsenal”).
  • News: use the suggested-source buttons, or paste any RSS feed URL.
  • Schedule: in the extension, set “free” windows (e.g. weekday evenings) when distractions are allowed; outside them they stay blocked.
  • Everything is private — your settings live only in your browser.
